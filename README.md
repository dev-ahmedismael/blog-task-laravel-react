# Blogger

This project contains a **Laravel backend** and a **React/Next.js frontend** running in Docker.  

After following these instructions, the backend will be accessible on **http://localhost:8000** and the frontend on **http://localhost:3000**.

---

## Setup & Run

1. Clone the repository:  

      git clone https://github.com/dev-ahmedismael/blog-task-laravel-react.git
   
      cd blog-task-laravel-react

 2. Build and run the project with Docker Compose:

      docker-compose up -d --build
    
 3. Open your browser and go to:

      Frontend: http://localhost:3000
    
      Backend: http://localhost:8000
